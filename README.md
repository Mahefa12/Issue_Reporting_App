# Citizen Issues Reporting Application

## What It Does

This application enables citizens to report issues (sanitation, roads, utilities, etc.) with detailed descriptions, location information, and file attachments. It features an interactive progress system that encourages user engagement.

## Working Features

- **Issue Reporting**: Complete form with location, category selection, description, and file attachments
- **Improvement Ideas**: Submit suggestions for community improvements

## How to Use

1. Start the app and select "Report Issues" from the main menu
2. Fill out the form:
   - Enter the issue location
   - Select a category (Sanitation, Roads, Utilities, etc.)
   - Provide a detailed description
   - Optionally attach files (images, documents)
3. Watch the progress bar fill as you complete fields
4. Submit your report and receive a confirmation with issue ID
5. Use "Improvement Ideas" to submit community enhancement suggestions

## How to Run

### Requirements
- Windows OS
- .NET 8.0 or higher

### Using Visual Studio
1. Open `IssuesReportingApp.sln`
2. Build the solution (`Ctrl+Shift+B`)
3. Run with `F5` or `Ctrl+F5` or `dotnet run` on the terminal

### Using Command Line
1. Navigate to the project directory
2. Run `dotnet build`
3. Run `dotnet run` or execute the built `.exe` file
